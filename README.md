# 🎉 AI Event Organizer and Scheduling System

## 📌 Project Overview
The AI Event Organizer and Scheduling System is a web application designed to simplify event planning and scheduling. It provides an interactive and user-friendly interface for managing events efficiently.

## 🚀 Features
- User Login
- Admin Dashboard
- Event Scheduling
- AI Event Assistant
- Responsive Design
- Modern User Interface

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- Python
- Git
- GitHub
